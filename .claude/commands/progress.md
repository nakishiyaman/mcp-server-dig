以下を確認して現在の開発状態をサマリー報告：

1. 現在のブランチとコミット状況: !`git status --short && git log --oneline -5`
2. 未コミットの変更ファイル一覧
3. `reference/HANDOFF.md` を読んで、前セッションの完了事項・次のタスク・ブロッカーを把握
4. `reference/ROADMAP.md` を読んで、全体の進捗を評価
5. 上記を統合して、直近で作業すべき次のタスクを提案

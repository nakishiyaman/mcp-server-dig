セッション終了時の引き継ぎ処理を行う。

0. **未コミット変更の処理**:
   - `git status` で未コミット変更を確認
   - mainブランチにいる場合 → 適切なブランチを作成して分岐
   - 変更をコミット・push（引き継ぎ記録自体もコミット・pushする）
1. `reference/ROADMAP.md` を読み、現在のタスク一覧を確認
2. 今回のセッションで完了したタスクのチェックボックスを `[x]` に更新
3. `git diff --stat` と `git log --oneline -5` で現在の変更状態を確認
4. 以下のフォーマットで引き継ぎサマリーを **`reference/HANDOFF.md` に書き出す**（上書き）:

```
## セッション引き継ぎ

日時: [今日の日付]

### 完了したタスク
- [今回完了した項目]

### 現在の状態
- ブランチ: [現在のブランチ]
- 未コミット変更: [あり/なし、変更ファイル一覧]

### 次にやるべきこと
- [次のタスク]

### ブロッカー/注意点
- [あれば記述]
```

5. `reference/ROADMAP.md` の「最終更新」日付を今日に更新
6. 画面にもサマリーを出力する
